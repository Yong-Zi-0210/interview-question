## 作用域

1、什么事作用域？
作用域：指程序源代码中定义变量的区域，它规定了如何查找变了，确定了当前执行代码对变量的访问权

2、js 是词法作用域（静态作用域），也就是说函数在声明的时候，它的作用域就已经确定了，不是调用的时候才确定（动态作用域）

## https = http + ssl/tls

# https 链接过程（TLS 链接过程）

1、tcp 三次握手完成后，开始进行 tls 链接
2、客户端向服务器发起链接请求： client hello 报文 该报文含有客户端上的的随机数、tls 协议版本、密码套件列表
3、服务端做出响应：server hello 报文，生成服务端随机数、确认协议版本（不支持则关闭机密通道）、确认加密套件；然后向客户端发送数字证书
4、服务端使用 CA 的公钥对数字证书进行检验->取出服务器的公钥匙，利用之前的自己生成的随机数和从服务端获取的随机数计算出第三窜随机数 pre-master、使用服务器的公钥对 pre-master 进行加密生成 pre-master key 发送给服务器，接着告诉服务器接下来会改变会话加密方式，使用新生成的会话密钥进行通信，客户端把之前的会话生成摘要，使用协商后的会话密钥进行发送给服务端，服务端能解出来说明会话密钥和协商的一至、自此客户端完成握手
5、服务端接受道 pre-master key 使用自己的私钥解密，取出第三串随机数，利用三个随机数生成会话密钥，告诉客户端改变通信加密方式，服务端也把之前的消息生成摘要使用新生成的会话密钥加密发送给客户端，客户端能够解出来说明协商成功。自此服务端握手完成
6、tcp 四次挥手
